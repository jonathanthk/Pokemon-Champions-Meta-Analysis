# Pokémon Champions Meta Analysis

## Overview
This project analyses in-game competitive Pokémon Champions battle data to identify trends in the current metagame. The dashboard enables users to explore 
Pokémon usage, move popularity, held items, abilities, natures, and seasonal meta changes through an interactive Tableau dashboard.

The goal of this project was to practice my skills in data analysis, dashboard design, and give me a competitive edge on Pokémon Champions. Made it to Masterball 2 :)

## Features
- Interactive dashboard
- Pokémon rankings
- Move and Typing based on weighted popularity
- Seasonal trends
